# 🛒 Shopping Cart Simulator (JSP)

A simple **Shopping Cart Simulator** web application built using **HTML, CSS, JSP, and Java**.
This project simulates a basic online shopping system where users can select products, enter quantities, and generate a bill automatically.

---

## 📌 Project Overview

The **Shopping Cart Simulator** demonstrates how a basic **e-commerce cart system** works.
Users can select items, specify quantities, and the system calculates the total cost using **JSP server-side processing**.

---

## 🚀 Features

* 🛍 Product listing with images
* 🔢 Quantity selection for each item
* 💰 Automatic price calculation
* 📄 Item-wise cost display
* 🧾 Total bill generation
* 🎨 Simple and user-friendly interface

---

## 🛠 Technologies Used

* **HTML** – Web page structure
* **CSS** – Styling and layout
* **JSP (Java Server Pages)** – Server-side processing

## ⚙️ How It Works

1. Open the **shopping page**.
2. Select the **quantity of products**.
3. Click the **Generate Bill** button.
4. The form sends data to `record.jsp` using the **POST method**.
5. JSP processes the request and **calculates the total bill**.
6. The generated bill is displayed with **item-wise cost and total amount**.
